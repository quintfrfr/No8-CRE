$$C_j = C_{A0}(\Theta_j + \nu_j \cdot X) P \left(\frac{T_0}{T}\right)$$ [cite: 4316]

Hence: [cite: 4322]

$$\Theta_A = \frac{F_{A0}}{F_{A0}} = 1 ; \Theta_B = \frac{2F_{A0}}{F_{A0}} = 2 ; \Theta_C = 0 ; \Theta_I = \frac{2F_{A0}}{F_{A0}} = 2$$ [cite: 4326] $$\nu_A = \nu_B = -1 ; \nu_C = 2$$ [cite: 4327]

Check for expansion factor ($\epsilon$) [cite: 4328]

$$A + B \rightleftharpoons 2C$$ [cite: 4329] $$\delta = 2 - (1+1) = 0, \quad \epsilon = y_{A0} \cdot \delta$$ [cite: 4330]

For equil molar of reactant, $y_{A0} = 0.5$; but $\delta = 0$. Hence $\epsilon = 0$. [cite: 4330]

"This capture that volumetric flow rate doesn't affected by $\epsilon$, solely affected by T&P." [cite: 4331]

Thus we have: [cite: 4332]

(c) Equilibrium Conversion [cite: 4349]

@ equilibrium, $-r_A'$ would be 0, hence: [cite: 4350]

$$0 = k(T) C_{A0}^2 \left[ (1-X)(2-X) - \frac{4X^2}{K_C(T)} \right] P^2 \left(\frac{T_0}{T}\right)^2$$ [cite: 4351]
