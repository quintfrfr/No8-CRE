Part (a) & (b) Balances & Equations

Mole balance for a packed bed reactor (PBR):

dX / dV = −r'_A / F_A0

Rate law for this elementary reversible reaction:

−r'_A = k(T) · [ C_A · C_B − C_C² / k_c(T) ]

From A + B ⇌ 2C, stoichiometric coefficients: a = 1, b = 1, c = 2

Expansion factor check — since δ = 2 − (1+1) = 0, and ε = y_A0 · δ = 0, volumetric flow rate is solely a function of T and P.

Θ_A = 1 ; Θ_B = F_B0/F_A0 = 2 ; Θ_C = 0 ν_A = ν_B = −1 ; ν_C = +2

Concentrations as functions of X, P, and T (with ε = 0):

C_A = C_A0 · (1 − X) · P · (T_0/T) C_B = C_A0 · (2 − X) · P · (T_0/T) C_C = C_A0 · (2X) · P · (T_0/T)

Substituting all concentrations into the rate law:

−r'_A = k(T) · C_A0² · [ (1−X)(2−X) − 4X² / k_c(T) ] · P² · (T_0/T)²

Temperature-dependent rate constants (Arrhenius / van't Hoff):

k(T) = 0.0002 · exp[ 25000/1.987 · (1/300 − 1/T) ] k_c(T) = 1000 · exp[ −20000/1.987 · (1/305 − 1/T) ]
Part (c) Equilibrium Conversion X_e vs Temperature

At equilibrium, −r'_A → 0, so the bracketed term equals zero:

(1 − X_e)(2 − X_e) − 4X_e² / k_c(T) = 0

Expanding and rearranging into quadratic form (× k_c/4):

(k_c/4 − 1) · X_e² − (3k_c/4) · X_e + (k_c/2) = 0

Applying the quadratic formula (a, b, c identified above):

X_e = [ 3k_c/4 ± √( (3k_c/4)² − 2k_c·(k_c/4 − 1) ) ] ───────────────────────────────────────────────── 2 · (k_c/4 − 1)

Part (e) Adiabatic Energy Balance

General energy balance (non-adiabatic):

T = T₀ + X · [ −ΔH_Rx(T_r) + ΔCp·(T_r − T₀) ] ──────────────────────────────────────────── ΣΘ_i·Cp_i + X·ΔCp

Since ΔCp = 0, the adiabatic energy balance simplifies to:

T = T₀ + X · (−ΔH_Rx) / ΣΘ_i·Cp_i

Substituting values (T₀ = 325 K, −ΔH_Rx = 20,000 cal/mol, ΣΘ_i·Cp_i = 96 cal/mol·K):

T = 325 + (20000 / 96) · X T = 325 + 208.33 · X

Part (f) Plot X_e, X, p, T vs Catalyst Weight W

For a packed bed with pressure drop (Ergun equation, dimensionless form). Since ε = 0:

dp/dW = − (α / 2p) · (T / T₀) where α = 0.00015 kg⁻¹

The full ODE system solved simultaneously (Python / scipy.integrate.solve_ivp):

# ODE system for adiabatic PBR with pressure drop def pbr_model(W, y): X = y[0] # Conversion p = y[1] # Dimensionless pressure (P/P0) # Algebraic — Energy & Thermodynamics T = 325.0 + 208.33 * X k = 0.0002 * np.exp((25000 / R) * (1/300 - 1/T)) Kc = 1000 * np.exp((-20000 / R) * (1/305 - 1/T)) # Rate Law ra = k * (Ca0**2) * ((1-X)*(2-X) - (4*X**2)/Kc) * (p**2) * ((T0/T)**2) # Differential Equations dXdW = ra_prime / Fa0 dpdW = -alpha * (T / T0) / (2 * p) return [dXdW, dpdW]

Part (g) Isothermal Operation — Heat Removal Profile

To maintain isothermal operation, the cooling jacket must perfectly match heat generated at every point. The local heat generated:

Local Heat Generated = (−r'_A) · (−ΔH_Rx)

A new differential equation tracks cumulative heat removed Q̇ along catalyst weight W:

dQ̇/dW = (−r'_A) · (−ΔH_Rx) = 20,000 cal/mol · (−r'_A)

Converting catalyst weight W to reactor volume V via bulk density ρ_b = 1.4 kg/dm³:

W = ρ_b · V → V = W / 1.4 [dm³]
