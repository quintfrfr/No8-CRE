1. The Problem Setup (A + B ⇌ 2C)

For the elementary gas-phase reaction, we establish our initial conditions and parameters based strictly on the textbook data (correcting the solutions manual's errors).

  • Feed Ratios: \(\Theta_A = 1\), \(\Theta_B = 2\), \(\Theta_I = 2\)
  • Expansion Factor: \(\delta = 2 - 1 - 1 = 0 \implies \epsilon = 0\)
  • Heat Capacities: \(\Sigma \Theta_i C_{P_i} = 20 + 2(20) + 2(18) = 96 \text{ cal/mol}\cdot\text{K}\)
  • Delta Cp: \(\Delta C_P = 2(20) - 20 - 20 = 0\)

2. Mole Balance & Rate Law

Using the stoichiometric table for a gas-phase reaction with pressure drop and temperature variation, the rate of disappearance is defined as:

\[ -r_A' = k(T) C_{A0}^2 \left[ (1-X)(2-X) - \frac{4X^2}{K_C(T)} \right] p^2 \left(\frac{T_0}{T}\right)^2 \]

Coupled with the differential mole balance for a Packed-Bed Reactor (PBR):

\[ \frac{dX}{dW} = \frac{-r_A'}{F_{A0}} \]

3. Equilibrium Conversion (\(X_e\))

At equilibrium, \(-r_A' = 0\). The driving force (the bracketed term in the rate law) must equal zero. Expanding and solving the quadratic yields the exact formula:

\[ X_e = \frac{\frac{3K_C}{4} - \sqrt{\left(\frac{3K_C}{4}\right)^2 - 2K_C\left(\frac{K_C}{4} - 1\right)}}{2\left(\frac{K_C}{4} - 1\right)} \]

4. Part (f) - Adiabatic Simulation (Python)

Because \(\Delta C_P = 0\), the adiabatic energy balance collapses to a beautiful linear relationship:

\[ T = T_0 + \frac{-\Delta H_{Rx}^\circ}{\Sigma \Theta_i C_{P_i}} X = 325 + 208.33 X \]

Instead of Polymath, we use Python's scipy.integrate.solve_ivp.

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# Constants
Fa0, Ca0, T0, alpha, R = 5.0, 0.2, 325.0, 0.00015, 1.987

def adiabatic_pbr(W, y):
    X, p = y
    T = 325.0 + 208.33 * X
    k = 0.0002 * np.exp((25000 / R) * (1/300 - 1/T))
    Kc = 1000 * np.exp((-20000 / R) * (1/305 - 1/T))
    
    ra_prime = k * (Ca0**2) * ((1 - X)*(2 - X) - (4 * X**2)/Kc) * (p**2) * ((T0/T)**2)
    
    dXdW = ra_prime / Fa0
    dpdW = -alpha * (T / T0) / (2 * p)
    return [dXdW, dpdW]

# Solve
sol = solve_ivp(adiabatic_pbr, [0, 4000], [0.0, 1.0], t_eval=np.linspace(0, 4000, 500))

5. Part (g) - Isothermal Operation

For isothermal operation, \(T\) is locked at 325 K. The temperature ratio vanishes, and we must calculate the cumulative heat removed \(\dot{Q}\) over the reactor volume.

\[ \frac{d\dot{Q}}{dW} = (-r_A')(-\Delta H_{Rx}^\circ) \] 
# Isothermal Constants
delta_H = -20000
rho_b = 1.4

# Constants k and Kc evaluated at 325K
k = 0.0002 * np.exp((25000 / R) * (1/300 - 1/325))
Kc = 1000 * np.exp((-20000 / R) * (1/305 - 1/325))

def isothermal_pbr(W, y):
    X, p, Q_rem = y
    
    ra_prime = k * (Ca0**2) * ((1 - X)*(2 - X) - (4 * X**2)/Kc) * (p**2)
    
    dXdW = ra_prime / Fa0
    dpdW = -alpha / (2 * p) 
    dQdW = ra_prime * (-delta_H) 
    return [dXdW, dpdW, dQdW]

# Solve
sol = solve_ivp(isothermal_pbr, [0, 4000], [0.0, 1.0, 0.0], t_eval=np.linspace(0, 4000, 500))
